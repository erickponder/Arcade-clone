*****Frogger game*****

Rules: 
- Play as a character and start at the bottom
- Object is to get across the screen and make it to the water
- If you are hit by a bug the game will reset and you will be sent back to the starting position
- If you safely make it across and into the water, an alert will appear saying you won

How to play:
- Double click the index.html file to run the game in the browser
- Use the up, down, left and right keys to move the player
- Refresh the browser at the end to play again!
